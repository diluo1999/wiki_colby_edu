The first character of a variable can only be a letter, an underscore(_) or a dollar sign($), and there can not be spaces in between the characters. Only letters, numbers, underscores, or dollar signs can be used for naming.

 

Variable Declaration:

Variables in JavaScript can be declared with 3 keywords:

  1. var: the scope of the variable is its execution context.
  2. let: the scope of the variable is constrained to whichever scope it is declared in.
  3. const: value cannot be modified; constrained to the scope it's declared in.

 

Identifier scoping:

Javascript has two scopes: global and local. Global scope means it can be accessed anywhere in the file, while function scope means it is constrained in the scope of the function.
