Di Luo : Di Luo's CS231 Project 6

Created by Di Luo, last modified on Apr 13, 2019

Description

In this project, I used my BSTMap to determine the frequencies of all the words in a given text document. All the data is stored in a binary search tree constructed by key value pairs, in which the key represents the word and the value represents the time that the key appears in the file. Binary search tree is a data structure storing items that keeps the keys in sorted order, so that lookup and other operations can use the principle of binary search, which is more efficient. I created a binary search tree that reads in the text, separates it into words, and stores the key and the frequency in the tree. Then I created two important functions, the first of which is word count file function that can go through the tree and create a file showing the total word count and the frequency of each word, and the second of which can read the word count file and recreate the BTS for further research. Finally, I processed the reddit comments from 2008 to 2015 and got the useful word count data, and the result proved my hypothesis that the total word count is proportional to processing time.


Solution

First I created the WordCounter class that holds a BTSMap and a variable twc that counts the total word count. Then I created an important function analyze(), which generates the word count for a file of words. It reads the files line-by-line and splits lines into words by splitting on anything that is not (^) (a-z or A-Z or 0-9 or '). Each valid word will be put into the binary search tree and increase the total word count. Some other functions are created, such as getTotalWordCount() that returns the total word count, and getUniqueWordCount() that returns the number of unique words.


After that I wrote the function writeWordCountFile() that writes the contents of theBSTMap to a word count file, and readWordCountFile() that writes a method that reads the contents of a word count file and reconstructs the fields of the WordCount object to build up the tree. And then I createWCTester class to test the ability of my functions to reconstruct the tree. In this class I used() analyze to load counttest.txt, writeWordCount() to write the results to counts_ct.txt, readWordCount() to load counts_ct.txt, and writeWordCount() to write counts_ct_v2.txt. The result is that the two files are the same.

Reflection

In this project, I learned how to construct a data structure and processing codes to deal with large quantities of data efficiently. Since the ability to identify word frequencies in a large corpus has been the basis for a number of different research projects in the digital social sciences, this project opened my mind on how to do it with my knowledge of computer science.  With this method, I can look at word usage over time in books and articles that reveal how culture and language change.
