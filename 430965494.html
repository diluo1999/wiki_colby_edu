Di Luo : Di Luo's CS231 Project 8

Created by Di Luo, last modified on Apr 28, 2019

Description

In this project, we implemented a priority queue and that uses a heap and a comparator, and then we used the priority queue to analyze trends in 8 years of Reddit posts. A priority queue is a dynamic data structure that always returns the entry in the queue with the highest priority. The queue we created this week is based on a heap data structure, which is a specialized tree-based data structure that is essentially an almost complete tree that satisfies the heap property: in a max heap, for any given node C, if P is a parent node of C, then the key (the value) of P is greater than or equal to the key of C. In a min heap, the key of P is less than or equal to the key of C. The node at the "top" of the heap (with no parents) is called the root node. To analyze the trends, firstly I created FindCommonWords class to find the most common words in each year. And secondly I created FindTrends class to look at the trends of specific words of my choice over the time period of the data. Finally I visualized the trends and conducted analysis based on the data and the visualization.


Solution

First, I created FindCommonWords class to read in a word count file and report the most common words. I used the WordCounter class from the last project and stored the key value pairs in the Hashmap. Then I used PQHeap class to store the pairs in the order from the highest word count to the lowest. Finally, I got the frequencies for the 10 most common words for each comment files. Extension 2: Two more lists

In the second extension, I created two more lists of 6 words. The first one contains olympics, beijing, london, soccer, spain, germany. Since Olympics happened in 2008 (Beijing) and 2012 (London), World Cup happened in 2010 and 2014 and Spain and Germany are the champions of 2010 and 2014, I want to see if the frequency is proportional to the popularity for these words. The result after analyzing them is shown below:

In general, we can see that there is no apparent relationship between the frequency of these words and the popularity. For Olympics, the frequencies are almost the same during these years. For Beijing, the situation is the same. For London, there is a change from 2012 to 2013, which conforms to the 2012 Olympics in London. For soccer, we cannot see any increase in frequency after each world cup. For Spain and Germany, the situation is the same as that of soccer.


The second list contains ai, 3g, 4g, python, iphone, ipad. The result is shown below:

We can see that AI's popularity was very high is 2008 but dropped from 2008 to 2012. After 2012, its popularity started increasing, which conforms to the development of AI in recent years. 3G's popularity increased from 2008 to 2010, during which 3G was popular and 4G hasn't been created. After 2010, since 4G was released in 2010, the popularity of 3G started decreasing. 4G became more and more popular from 2008 to 2011, and excess that of 3G in 2012, but something strange happens that its popularity dropped from 2011 to 2015, which maybe interpreted that there was less interest in the mobile network in the US.

For python, its frequency was very big in 2008 but it suddenly decreased, then from 2009 to 2015 its popularity is stable.

For ipad, its popularity is close to 0 from before 2010 since it hadn't been released. Then in 2010 it was very popular, which conforms to the release of ipad 1. Then its popularity decreased in the following years.

For ipod, we can see that from 2009 to 2015, its popularity was continuously decreasing, which conforms to my knowledge that Apple focused less and less on ipod and more on iphone and ipad for the last ten years.
